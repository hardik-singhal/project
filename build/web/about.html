Our Stories

As a continuous effort to improve and increase the use of computers in students education, we implemented an online assessment server.

Its purpose is to ease the students examination process and to provide a transparent and objective evaluation.

  • Learning program with after-school
  • Positive learning environment
  • Learning through play

Our Solution

The solution we developed is a webapplication, written using PHP and Javascript.

It uses MySQL as a database management systemand Apache for the web server, hosted on a Linux system.

  • web browser capable of handling AJAX queries
  • can create tests and set its parameters
  • grade is automatically displayed
